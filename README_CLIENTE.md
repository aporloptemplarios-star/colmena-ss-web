# COLMENA WorkSuite Cliente

## Activacion

1. Abrir el launcher.
2. Iniciar sesion con usuario autorizado.
3. Introducir:
   - Nombre cliente / servidor.
   - Server ID.
   - License Key.
4. Pulsar `ACTIVAR LICENCIA`.

## Planes

### BASIC

Incluye:
- Launcher basico.
- Logs simples.
- Conexion Discord.
- Soporte basico.

No incluye:
- IA avanzada.
- COLMENA-SS completo.
- Anticheat avanzado.
- Dashboard.

### PREMIUM

Incluye:
- Launcher completo.
- IA basica.
- Logs avanzados.
- Anticheat activo.
- COLMENA-SS funcional.
- Soporte.

No incluye:
- Multi-servidor.
- Funciones enterprise.
- Analisis avanzado.

### ENTERPRISE_DIAMOND

Incluye todo:
- IA avanzada.
- COLMENA-SS PRO.
- Anticheat completo.
- Logs y auditoria.
- Multi-servidor.
- Prioridad.
- Personalizacion.

## Licencias demo para pruebas internas

```text
COLMENA-BASIC-DEMO
COLMENA-PREMIUM-DEMO
COLMENA-DIAMOND-DEMO
```

Estas claves deben reemplazarse por licencias reales en produccion.

## Seguridad

- No compartas tu license key.
- No publiques logs con datos sensibles.
- Cada cliente debe usar su propio `server_id`.
