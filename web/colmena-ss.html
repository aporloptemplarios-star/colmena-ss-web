Que es

Un servicio operativo, no solo una herramienta.

COLMENA-SS combina Discord, bot, launcher, IA de apoyo, logs y staff autorizado para que cada revision tenga orden, consentimiento y trazabilidad.

Para servidores

RedM/FiveM, roleplay, comunidades whitelist y equipos con staff activo.

Para clientes

Acceso Discord, rol segun plan, soporte, creditos de escaneo y reportes.

Como funciona

Un flujo simple para casos delicados.

01

Compra

El servidor contrata un escaneo individual o plan mensual.

02

Verificacion

Stripe confirma el pago y el bot vincula el Discord ID registrado.

03

Solicitud

El cliente abre la solicitud desde el canal de escaneos.

04

Revision

El usuario acepta, el staff revisa y se archivan evidencias autorizadas.

05

Resultado

El staff recibe un reporte final claro y util para tomar decisiones.

Privacidad

Controlado, consentido y con minimo dato necesario.

Consentimiento

No se realiza escaneo sin aceptacion del usuario y sin ticket trazable.

Minimo necesario

Se recopila solo informacion tecnica necesaria para la revision.

Staff autorizado

Evidencias, logs y reportes quedan restringidos a roles autorizados.

IA de apoyo

La IA recomienda y resume; no sanciona ni decide por el staff.

Planes COLMENA-SS

Compra por caso o activa servicio mensual.

Contratacion

No se entrega acceso sin pago validado y Discord ID asociado.
